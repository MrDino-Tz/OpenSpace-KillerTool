# OpenSpace-KillerTools

Welcome to OpenSpace-KillerTools, a collection of powerful web-based developer tools!

## Features (Planned)
- PDF Converter & Editor
- Markdown Generator with Conversions
- Image & Video Editor
- Emoji Picker
- ASCII Word Art Generator
- ...and more!

## Tech Stack
- React 18
- Vite
- Material UI (MUI)
- WebAssembly (for client-side processing)

## Running Locally

1. Install dependencies
```bash
npm install
```

2. Start the dev server
```bash
npm start
```
